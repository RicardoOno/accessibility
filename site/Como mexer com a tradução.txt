1° Faça um caminho igual a esse

|-esportes // seu tema
| |-index.php //index do tema
| |-noticia1.php  //noticia 1.

2° Na Index.php mude:
De:   $idioma = $idioma->BuscaIdioma('esportes', 'index');
Para: $idioma = $idioma->BuscaIdioma('[tema]', 'index');

Obs: [tema] -> É o seu tema e;
     'index' -> será onde a tradução será puxada. 
		Para ver a diferença, abra o noticia1.php e veja que esta:
		$idioma = $idioma->BuscaIdioma('esportes', 'not1');
		Ele ira dizer de onde o php ira pegar para fazer a tradução.

Ou seja: Voce ira apontar o tema e depois qual "parte" é para puxar do idioma.php

3° Idioma.php - Aqui é onde ocorre as traduções
'[tema]' => [
	 	'[variavel]' => '[Texto]',
	]

Ex: 
'esporte' => [
	 	'temaName' => 'Esporte',
		'home' => 'Página Inicial',
	]

'pt' => [ //traduções em portugues
     'esporte' => [ //tema
	'index' => [ //qual sera puxado. Lembre 'index'
		'temaName' => 'Esporte',
		'home' => 'Página Inicial',
	],
	'not1' => [
		'titulo' => 'Brasil perde de 7x1'
	]
     ]
], 

'en' => [ //traduções em ingles
     'esporte' => [
	'index' => [
		'temaName' => 'Sporte',
		'home' => 'Home',
	],
     ]
], 

'es' => [ //traduções em espanhol
     'esporte' => [ //tema
	'index' => [ //qual sera puxado. Lembre 'index'
		'temaName' => 'Deporte',
		'home' => 'Página de inicio',
	],
     ]
] 

É assim que a tradução e a seleção de qual sera traduzido funciona.

Mas pra sorte de voces, é so copiar o meu e mudar o '[Texto]'
